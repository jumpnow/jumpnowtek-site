Services

Founded in 2004, Jumpnow Tech is an embedded Linux consulting company providing a range of services.

Product Development

Embedded Linux development including:

  • Custom Linux systems using tools from either Yocto / OpenEmbedded or Buildroot
  • Boot loader customization (u-boot)
  • Kernel selection and customization including device tree files
  • Device drivers for new hardware
  • General application development in C, C++ and Python using standard middleware tools like MQTT
  • Cross-platform Qt software development including GUI touchscreen applications
  • Small web applications for configuration and monitoring using Python Flask

Vulnerability Assessments

Network and physical testing of embedded / IoT devices whether Linux or Windows based.

Mender Integration

Mender is an open source system for updating embedded Linux devices. Mender uses an A/B rootfs approach that provides safe, atomic upgrades with reliable fallback when required.

Jumpnow has experience integrating Mender on a number of customer projects including products based on the hardware platforms listed below.

  • Raspberry Pi (RPi3, RPi0w, CM3)
  • Beaglebone (all flavors)
  • i.MX6 boards from various manufacturers
