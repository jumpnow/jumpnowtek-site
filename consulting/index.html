Mender Integration

Mender is an open source system for updating embedded Linux devices. Mender uses an A/B rootfs approach that provides safe, atomic upgrades with reliable fallback when required.

Jumpnow has experience integrating Mender on a number of customer projects including all of the hardware platforms listed below.


Common Hardware Platforms


Raspberry Pi

There are a number of Rasberry Pi boards to choose from with the Raspberry Pi Compute in particular targeted at commercial applications using customized carrier boards. The two most popular boards for customer projects have been the RPi3-B and the RPi0-W.

Gumstix is one company selling a reference Pi Compute Dev Board and has a service where you can design your own Pi Compute board using their Gepetto online design tool.

Beaglebone Black

Based on the TI Sitara AM335x processor, the BeagleBones are a popular family of boards for hobbyists and commercial projects.

The boards are well supported by the mainline Linux kernel and U-Boot bootloader projects.

NXP i.MX6

The NXP i.MX6 processors are based on the ARM Cortex-A9 architecture and come in single, dual and quad-core packages. They run Linux and are suitable for many applications ranging from robotics to multimedia.

There are multiple vendors for i.MX6 hardware.

Gumstix

Gumstix® are small, powerful, computer-on-module systems designed for embedded and robotics applications. The Gumstix Overo® COMs are based on the Texas Instruments OMAP3 and Digital Media ARM Cortex-A8 processors. The Gumstix Duovero® COMs are based on the TI OMAP4 Cortex-A9 dual-core processors.

Gumstix systems run Linux and many open-source software programs. Custom software applications can be written in C/C++, Python, Perl, Ruby or any language you normally develop with on Linux.

Jumpnow Technologies is a Verified Design Consultant for Gumstix products.
