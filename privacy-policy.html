Privacy Policy

Jumpnow Technologies, LLC operates the website https://jumpnowtek.com.

Information Collection and Use

Personal information is not intentionally collected other then what can be derived from standard web server logging as described below.

The website does not use cookies, javascript or utilize any third-party tracking services.

Logging

Web server logging is enabled for standard maintenance: usage stats, bad links, malicious request monitoring.

The following data is collected and retained for 14 days before deletion

  • IP address
  • Date and time of access
  • Page requested
  • Browser User Agent

Log data is not shared with third-parties.

Third-party Links

Third-party links are used throughout the website. If you click on a third-party link, you will be directed to that site. We have no control over the privacy policy of those sites.
